Phishing

El phishing es una técnica de ciberdelincuencia que busca obtener información confidencial, como contraseñas, números de tarjetas de crédito y datos personales, haciéndose pasar por entidades legítimas a través de correos electrónicos, mensajes o sitios web fraudulentos. Los atacantes utilizan tácticas de ingeniería social para engañar a las víctimas y que revelen su información, o para que instalen software malicioso.

Ransomware

El ransomware es un tipo de malware que retiene como rehenes los datos confidenciales o el dispositivo de una víctima, amenazando con mantenerlos bloqueados, o algo peor, a menos que la víctima pague un rescate al atacante.

Malware

El malware es software malicioso diseñado para infiltrarse en sistemas informáticos con el objetivo de causar daño, robar información o tomar el control del sistema. Existen diversos tipos de malware, como virus, gusanos, troyanos, spyware y ransomware, cada uno con sus propios métodos de propagación y objetivos.

Ataque DDoS

Un ataque DDoS (Denegación de Servicio Distribuido) es un ciberataque que busca sobrecargar un servidor o red con tráfico malicioso, impidiendo que los usuarios legítimos puedan acceder a los servicios. Este tipo de ataque se diferencia de un ataque DoS en que utiliza múltiples fuentes para generar el tráfico, a menudo una botnet de dispositivos infectados, mientras que un ataque DoS proviene de una sola fuente.
